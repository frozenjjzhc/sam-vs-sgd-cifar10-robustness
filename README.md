# SAM vs SGD CIFAR-10 Robustness

This repository contains a Jupyter notebook comparing the robustness of Sharpness-Aware Minimization (SAM) and standard SGD on CIFAR-10.

## Files

- `SAM_vs_SGD_CIFAR10_Robustness.ipynb`: main experiment notebook

## Notes

- The notebook is stored as a regular `.ipynb` file and can be opened with Jupyter Notebook, JupyterLab, or VS Code.
